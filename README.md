# 脑洞填补器 (多智能体白皮书工坊 v4)

多智能体推演工具：输入需求，让 AI（架构师、挑刺者、主持人）自动闭门讨论，生成可落地的技术白皮书。

### v4 核心升级
- **LangGraph 引擎**：重构底层状态机，长对话更稳定，杜绝死循环。
- **动态退火**：前期严格挑刺，后期强制收敛，兼顾方案深度与执行效率。
- **长程记忆**：跨轮自动摘要，防止 LLM 遗忘历史上下文。
- **独立 UI 线程**：打字机流式输出，双 Buffer 架构防 Token 爆炸。

### 快速启动

1. **克隆项目**
   ```bash
   git clone https://github.com/assummarry/ai-meeting.git
   cd ai-meeting
   ```

2. **启动** (首次运行会自动配置环境和依赖)
   - **Mac**: 双击 `mac系统运行.command`
   - **Windows**: 双击 `win系统双击打开.bat`

3. **配置**
   在自动弹出的浏览器页面侧边栏，填入 API Key 即可开始推演。
